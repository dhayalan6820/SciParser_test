# GEMINI COST ENGINE

The application currently uses:

`google/gemini-3-flash-preview`

Do not hardcode pricing values.

Instead:

1. Create a model pricing configuration table.
2. Version pricing records by effective date.
3. Resolve pricing dynamically based on the model and request timestamp.
4. Calculate input and output costs independently.
5. Store the exact pricing version used for each request to preserve historical accuracy when Google changes pricing.
