Create a modern, production-ready Admin Dashboard for an AI SaaS platform using React, TypeScript, Tailwind CSS, and shadcn/ui.

The design should be premium, clean, responsive, and optimized for desktop, tablet, and mobile.

Follow modern SaaS dashboards like Vercel, Stripe, OpenAI, Linear, Notion, and Supabase.

==================================================
Design Requirements
==================================================

• Minimal UI
• Dark and Light theme
• Soft shadows
• Rounded corners (12-16px)
• Excellent spacing
• Smooth animations
• Glassmorphism where appropriate
• Skeleton loaders
• Empty states
• Loading states
• Error states
• Hover animations
• Responsive layout
• Accessible (WCAG)
