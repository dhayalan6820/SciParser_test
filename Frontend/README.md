# SciParser Frontend

The frontend of SciParser is a modern React application providing a seamless interface for interacting with AI agents.

## Tech Stack
- **React**: UI library.
- **Vite**: Build tool.
- **Tailwind CSS**: Styling.
- **Shadcn UI**: Component library.
- **Framer Motion**: Animations.

## Setup

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Environment Variables**:
   Create a `.env` file in the `Frontend/` directory:
   ```env
   VITE_API_URL=http://localhost:8000
   ```

3. **Run Development Server**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```

## Features
- **Chat Interface**: Real-time communication with agents.
- **Agent Plan Visualization**: See the step-by-step reasoning of the AI.
- **Browser Preview**: Live stream of the agent's web interactions.
- **Responsive Design**: Optimized for various screen sizes.
